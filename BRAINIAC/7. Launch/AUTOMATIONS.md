# Automations, Skills, and Tools: Launch Phase

Amplify your reach and perfectly coordinate release day.

## 1. Product Hunt Checklist Automation
- **Tool**: Notion / Linear API sync
- **Use Case**: Never miss a step on launch day by auto-generating the launch timeline and assigning tasks.
- **Skill Implementation**: A script that outputs a prioritized checklist including maker comments, taglines, assets ready, and the exact release hour countdown.

## 2. Social Media Crossposter
- **Tool**: Typefully / Buffer API integrations
- **Use Case**: Simultaneously publish your launch announcement across X (Twitter), LinkedIn, and IndieHackers.
- **Skill Implementation**: An automated workflow that takes your core launch message and adapts/schedules it natively for each platform to maximize algorithm visibility.
