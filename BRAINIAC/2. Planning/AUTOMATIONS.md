# Automations, Skills, and Tools: Planning Phase

Keep project scope tight and momentum high.

## 1. Automated Ticket Creation
- **Tool**: Linear / Jira API scripts
- **Use Case**: Automatically translate the `MVP Scope` markdown file into epics and actionable tickets.
- **Skill Implementation**: A quick sync script that reads `todo.md` or roadmap artifacts and updates task states in your external issue tracker.

## 2. Roadmap Prioritization Skills
- **Tool**: Local LLM / Analytical Prompts
- **Use Case**: Use the RICE score framework (Reach, Impact, Confidence, Effort) to rank features.
- **Skill Implementation**: An AI prompt that ingests the proposed features list and outputs a sorted RICE priority table.
