# Automations, Skills, and Tools: Design Phase

Accelerate the transition from raw ideas to functional UI.

## 1. Figma-to-Code Scripts
- **Tool**: v0 by Vercel / Figma REST API / Relume AI
- **Use Case**: Automatically scaffold UI components directly from design layouts or text descriptions.
- **Skill Implementation**: A workflow that generates the initial component structure, CSS modules, and React/Tailwind boilerplate matching your design system variables.

## 2. UI/UX Accessibility Checkers
- **Tool**: Axe Core CLI / Lighthouse
- **Use Case**: Perform rapid CI-based accessibility contrast and markup validation before coding logic.
- **Skill Implementation**: A quick script that runs accessibility tests on visual prototypes to catch contrast and structural issues early.
