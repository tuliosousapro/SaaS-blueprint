# Automations, Skills, and Tools: Testing Phase

Catch regressions before they reach your users.

## 1. E2E Automation Scripts
- **Tool**: Playwright / Cypress AI Generators
- **Use Case**: Simulate real user flows, like signup to checkout, automatically on every build.
- **Skill Implementation**: Use AI prompts that read a screen's HTML and generate bulletproof Playwright selectors and assertions for critical user paths.

## 2. API Load Testing Tools
- **Tool**: K6 / Artillery scripts
- **Use Case**: Ensure the application scales under pressure before launch day.
- **Skill Implementation**: Simple scripts that simulate 1,000 concurrent users hitting your main endpoints, exposing bottlenecks in queries or hosting configuration.
