# Automations, Skills, and Tools: Idea Phase

Automate early-stage research to validate problems before building.

## 1. Market Research Scraper Scripts
- **Tool**: Firecrawl MCP (`firecrawl_scrape`, `firecrawl_crawl`)
- **Use Case**: Automatically scrape Reddit, Quora, and niche forums for recurring complaints and keywords to identify organic market gaps.
- **Skill Implementation**: Create a subagent instruction to navigate target communities and output structured JSON pain-point summaries.

## 2. Competitor Analysis Skills
- **Tool**: Standard Search MCP / Browser Automation
- **Use Case**: Automatically build a feature matrix of top 3 competitors, including pricing and missing features.
- **Skill Implementation**: Use an AI template to extract positioning statements and pricing models directly from competitor landing pages into a markdown artifact.
