# Automations, Skills, and Tools: Growth Phase

Build systems that multiply every user into more users automatically.

## 1. Referral Loop / Affiliate Program Generators
- **Tool**: Rewardful / Tapfiliate API Integrations
- **Use Case**: Instantly generate and track custom affiliate links whenever a new user signs up.
- **Skill Implementation**: A backend hook that auto-registers every new client as an affiliate, appending their unique referral string to a shareable dashboard widget.

## 2. PLG (Product-Led Growth) Metric Trackers
- **Tool**: K-Factor simulation scripts
- **Use Case**: Understand the exact virality coefficient of your application features.
- **Skill Implementation**: A SQL-based cohort analysis script that isolates how many net-new invites are sent per existing active user per month.
